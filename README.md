# car-dodge
2D puzzle game made in c that runs in terminal

# How to run:
    ./traffic <name of map file eg. map.txt>

# How to play:
*Keyboard only!*

## movement
w - up
a - left
s - down
d - right
u - undo movement

## game
P - player
G - goal *move to this thing!*
\> - car *dodge this thing!*

# Making your own map:
Look at map.txt for an example!
